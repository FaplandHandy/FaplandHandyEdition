Welcome to Fapland: Handy Edition. 
To get started:
	1) Generate your ApplicationID from https://user.handyfeeling.com/ (create an account and then generate new applicationID (note: generate applicationID not 			applicationKey)
	2) Install VLC and Python (make sure Python is added to PATH ie. when you type python in Powershell you shouldn't see command not recognized).
	3) Make sure your Handy is updated to firmware 4 (use https://new.handyfeeling.com/ to update)
	4) Enter your applicationID and handy connection key in the handy settings on the game title screen
	5) Install python dependencies using the button on title screen
	6) Make sure you downloaded the media folder from the mega and is next to the exe
	7) Start the game!

Game details:

You have to start from Round 1 and go up to Round 100 (Final Round) by rolling the dice.
There are also perks:
S class : Obtain one randomly after rounds 25,50,75
	Lucky 7: Next dice roll is guaranteed to be 7
	Skip round

A class : Obtain one with 10% base chance after each round
	Dice reroll
	Huge dice: Make dice range 1-10 for next 2 rounds

B class : Obtain one with 15% base chance after each round
	Bonus pauses: Get +3 video pauses added
	Pause extension: Double the pause duration for next 3 rounds (10s)


If you have 5 perks and get a perk again after a round, you lose the earliest perk obtained.

Video Controls:
	Press E if you cum (Game over)
	Press R to resync Handy if it's not moving or out of sync
	Press SPC to use your pauses
	Press Q to close the video (cheat)
